# OrienteTech
Charla de bienvenida dirigida a distintos perfiles del campo IT.

## Dependencias
* nodejs
* npm
* bower
* gulp
* browser-sync

### Traer dependencias del proyecto

	npm install

### Traer dependencias front-end

	bower install

### Ejecutar servidor web local y ver los cambios en tiempo real

	gulp
